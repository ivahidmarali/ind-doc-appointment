# IND Doc Appointment Finder
A simple NodeJS app to find sweet IND Doc pickup appointment.

## Installation
```shell
git clone git@github.com:mrcyna/ind-doc-appointment.git ind-doc-appointment
cd ind-doc-appointment
npm install
npm start
```
